# AWS Static Website Hosting Project

This project demonstrates how to deploy a fully static website using **Amazon S3** and make it publicly accessible through S3 Static Website Hosting.

---

## 🚀 Project Overview
This project includes:

- Creating a public S3 bucket
- Uploading static website files (`index.html`, `styles.css`)
- Enabling S3 Static Website Hosting
- Applying the correct bucket policy
- Accessing the live website via the S3 website endpoint

---

## 📁 Tech Stack
- HTML / CSS  
- Amazon S3  
- Static Website Hosting  
- Git & GitHub
